Size: 3x3. Calibration data for the accelerometer. This is not reported by the Wii Remote directly - it is instead collected from normal Wii Remote accelerometer data.

See also
AccelCalibrationStep, CalibrateAccel(AccelCalibrationStep)

Here are the 3 calibration steps:

  1. Horizontal with the A button facing up
  2. IR sensor down on the table so the expansion port is facing up
  3. Laying on its side, so the left side is facing up

By default this is set to experimental calibration data.

int[calibration step,calibration data] (size 3x3)

Property Documentation

ReadOnlyArray<int> WiimoteApi.AccelData.accel
get

Current remote-space acceleration, in the Wii Remote's coordinate system. These are RAW values, so they are not with respect to a zero point. See CalibrateAccel(). This is only updated if the Wii Remote has a report mode that supports the Accelerometer.

Warning
This should not be used unless if you want to calibrate the accelerometer manually. Use CalibrateAccel() instead.

Range: 0 - 1024
The sign of the directions below are with respect to the zero point of the accelerometer:
Up/Down: +Z/-Z
Left/Right: +X/-X
Forward/Backward: -Y/+Y
