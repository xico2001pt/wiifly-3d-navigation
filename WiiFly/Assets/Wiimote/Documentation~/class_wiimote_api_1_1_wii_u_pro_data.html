ReadOnlyArray<ushort> WiimoteApi.WiiUProData.lstick
get

Pro Controller left stick analog values. This is a size-2 array [X,Y] of RAW (unprocessed) stick data. These values are in the range 803-3225 in the X direction and 843-3291 in the Y direction.

Note
Min/Max values may vary between controllers (untested). One way to calibrate is to prompt the user to spin the control sticks in circles and record the min/max values.
