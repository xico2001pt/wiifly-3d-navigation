float [] WiimoteApi.IRData.GetPointingPosition ( )

Size: 2. Returns the position at which the Wii Remote is pointing to. This is a value from 0-1 representing the camera-space pointing position in X and Y. Assume a 4x3 aspect ratio.

This takes into account the rotation of the remote (using the Wii Remote's Accelerometer) to correct for rotational distortion.

float [,] WiimoteApi.IRData.GetProbableSensorBarIR ( bool  predict = true)

Attempts to identify which of the four IR "dots" reported by the Wii Remote are from the Wii sensor bar.

Parameters
predictIf true, and one of the dots is outside of the Wii Remote's field of view, WiimoteApi will attempt to predict the other dot's position outside of the screen (default true).
Returns
First Dimension: Index of detected IR dot.
Second Dimension: 0: X, 1: Y, 2: Index in ir (or -1 if predicted)
Size: 2x3
Range: 0-1 with respect to the Wii Remote Camera dimensions. If predict is true this may be outside of that range.

Size: 4x3. Current Wii Remote RAW IR data. Wii Remote IR data can detect up to four IR dots. Data = -1 if it is inapplicable (for example, if there are less than four dots, or if size data is unavailable due to the selected IRDataType).
